# E2E Action Execution Testing

## Pre-requisites

- `cargo tauri dev` running
- Razer Naga V2 HyperSpeed connected
- Runtime started (green dot in sidebar)
- Notepad open as target window

---

## Phase 1: Shortcuts (already assigned)

Open Notepad, type some text, then test each thumb button.

| # | Button | Expected | Result | Notes |
|---|--------|----------|--------|-------|
| 1 | Thumb with Ctrl+C | Copy selected text | | |
| 2 | Thumb with Ctrl+V | Paste from clipboard | | |
| 3 | Thumb with Ctrl+Z | Undo last action | | |
| 4 | Thumb with Ctrl+S | Save dialog opens | | |
| 5 | Thumb with Ctrl+F | Find dialog opens | | |
| 6 | Thumb with Delete | Delete selected text | | |
| 7 | Any other shortcut | As assigned | | |

**Edge case:** Hold Ctrl physically, then press a Ctrl+X button. Expected: error in debug log ("external modifier already pressed").

---

## Phase 2: TextSnippet — SendText

1. In Expert mode, assign a thumb button to TextSnippet
2. Set text: `Hello World 123!`
3. Set paste mode: **Direct text input** (SendText)
4. Save config (Ctrl+S)
5. Focus Notepad
6. Press the button

| Check | Expected | Result |
|-------|----------|--------|
| Text appears in Notepad | `Hello World 123!` typed character by character | |
| Cyrillic text | Assign `Привет мир!`, test same way | |
| Multiline text | Assign `Line1\nLine2`, verify two lines | |
| Empty text | Assign empty string, verify no crash | |

---

## Phase 3: TextSnippet — ClipboardPaste

1. Change the same button's paste mode to **Clipboard paste** (ClipboardPaste)
2. Save config
3. Copy something to clipboard first (e.g. "ORIGINAL")
4. Focus Notepad
5. Press the button

| Check | Expected | Result |
|-------|----------|--------|
| Text pasted | `Hello World 123!` appears instantly | |
| Clipboard restored | After 200ms, Ctrl+V pastes "ORIGINAL" again | |
| Cyrillic via clipboard | Assign `Привет мир!`, verify correct encoding | |

## Phase 4: Launch

1. Assign a thumb button to Launch action
2. Set target: `C:\Windows\System32\calc.exe`
3. Save config
4. Press the button

| Check | Expected | Result |
|-------|----------|--------|
| Calculator opens | calc.exe spawns | |
| Second press | Another calc instance opens | |
| Invalid path | Set target to `C:\nonexistent.exe`, verify error in debug log | |

---

## Phase 5: Sequence (Macro)

1. Assign a thumb button to Sequence
2. Add steps:
   - Step 1: Send `Ctrl+A` (select all)
   - Step 2: Sleep 200ms
   - Step 3: Send `Ctrl+C` (copy)
   - Step 4: Sleep 100ms
   - Step 5: Text `Copied!`
3. Save config
4. Type text in Notepad, then press the button

| Check | Expected | Result |
|-------|----------|--------|
| Full sequence runs | Text selected, copied, "Copied!" typed | |
| Delay respected | Visible pause between steps | |
| Partial failure | Remove text from step 5, add invalid step, verify error handling | |

## Phase 6: Edge Cases

### 6a. Elevated Window (WIN-001)

1. Open CMD as Administrator (right-click → Run as admin)
2. Try shortcuts in the elevated CMD
3. Try TextSnippet in the elevated CMD

| Check | Expected | Result |
|-------|----------|--------|
| Shortcut in elevated CMD | May fail (UIPI blocks SendInput) | |
| TextSnippet in elevated CMD | May fail | |
| Error reported | Debug log shows UIPI error | |

### 6b. Clipboard Contention (WIN-002)

1. Open a clipboard manager (if installed) or keep clipboard history (Win+V)
2. Copy something
3. Press ClipboardPaste button
4. Immediately check clipboard (Ctrl+V somewhere)

| Check | Expected | Result |
|-------|----------|--------|
| Text pasted correctly | Yes | |
| Original clipboard restored | Within ~200ms | |
| Rapid double-press | Press button twice quickly, verify no crash | |

### 6c. Self-window Detection

1. Focus the Naga Studio window itself
2. Press any assigned button

| Check | Expected | Result |
|-------|----------|--------|
| Action NOT executed | Window is ignored (self-detection) | |
| Debug log shows "ignored" | Yes | |

### 6d. Hypershift Layer

1. Switch to Hypershift tab
2. Assign different actions to some buttons
3. Hold Hypershift button + press thumb button

| Check | Expected | Result |
|-------|----------|--------|
| Hypershift action fires | Not the standard layer action | |
| Release Hypershift | Standard actions resume | |
